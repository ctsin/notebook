# Launch with sandbox

```sh
gemini --sandbox
```

# Check pointing

```sh
gemini --checkpointinig
```

# Save Chat

```sh
/chat save <A_NAME>

/chat resume <A_NAME>
```

# setup Gemini API Key in project base

1. Create a `.env` file in the project root.
2. Add the following line to the `.env` file: `GEMINI_API_KEY=<YOUR_GEMINI_API_KEY>`
